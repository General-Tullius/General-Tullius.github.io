2021-02-01 Linux

  1. 查看登录日志

    # last -n
last -5

  2. 查看系统运行资源状况

    top

  3. 查看系统负载

    uptime

  1. SSH登录

    # ssh -p port username@host
ssh user@***.***.***.***

  2. SSH传输文件

    # 从服务器获取文件
# scp -p port username@host:/服务器路径 /本地路径
scp -P 666 user@***.***.***.***:filePath localPath

# 上传文件到服务器
# scp -p port /本地路径 username@host:/服务器路径 
scp -P 666  localpath user@***.***.***.***:filePath
