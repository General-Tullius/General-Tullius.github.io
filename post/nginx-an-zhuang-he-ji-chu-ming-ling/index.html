nginx安装和基础命令

2021-02-01 nginx

安装

Windows：

# https://nginx.org/en/download.html
# 下载合适的版本，解压放在合适的磁盘位置

Mac：

$ brew install nginx

基础命令

1. 启动

# Windows
# cd 至 nginx 根目录
# 后台运行
$ start .\nginx
# 临时运行
$ .\nginx

# Mac
# 后台运行并开机自启动
$ brew services start nginx
# 临时运行
$ nginx

2. 维护

# 重载配置
# 重新加载配置文件
# 不中断nginx服务，conf配置有错，会用上一次正确配置正常运行。
$ nginx -s reload

# 重启
# nginx服务会中断，若conf配置有误，则nginx不能启动
$ nginx - s restart

# 截断日志
# 用于防止日志文件过大,需要先移动文件，然后执行此命令
$ nginx -s reopen

3. 停止

 # 快速停止
 # 可能并不保存相关信息
 nginx -s stop
 
 # 有序停止
 # 完整有序停止，并保存相关信息
 nginx -s quit
